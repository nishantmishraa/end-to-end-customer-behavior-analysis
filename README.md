# 📊 Customer Behavior Analytics Dashboard

An interactive Power BI dashboard designed to analyze customer behavior, purchasing patterns, and key business metrics. This project transforms raw customer data into actionable insights through dynamic visualizations and KPI tracking.

---

## 🚀 Project Overview

The goal of this project is to help businesses understand their customers by analyzing purchasing behavior, customer segmentation, revenue trends, and performance metrics.

Using Power BI, the dashboard enables users to explore data interactively and make data-driven decisions.

---

## 📌 Features

* Interactive Power BI Dashboard
* Customer Segmentation Analysis
* Sales & Revenue Tracking
* Customer Purchase Trends
* KPI Cards for Business Metrics
* Dynamic Filters and Slicers
* Clean and User-Friendly Dashboard Design

---

## 🛠️ Tools & Technologies

* **Power BI**
* **Power Query**
